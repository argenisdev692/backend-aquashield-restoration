# 🎬 Campaigns Module — Video Production Guide

> **Módulo genérico NestJS** · Hexagonal + DDD · TOFU · MOFU · BOFU · LOYALTY
> Formatos: 9:16 (Vertical) y 16:9 (Horizontal) · Duración: 15–20 seg

---

## Etapas del Funnel

| Etapa | Objetivo | Audiencia | Tono |
|-------|----------|-----------|------|
| **TOFU** | Awareness | Fría – nunca ha oído de la marca | Educativo, empático |
| **MOFU** | Consideración | Tibia – conoce el problema | Informativo, confianza |
| **BOFU** | Conversión | Caliente – lista para actuar | Urgente, directo |
| **LOYALTY** | Retención / Referidos | Clientes existentes | Calidez, comunidad |

---

## TOFU — Top of Funnel — Awareness

**Objetivo:** Generar conocimiento de marca y del problema. Audiencia fría que nunca ha oído del negocio.  
**Audiencia:** Audiencia fría en Meta/TikTok segmentada por ubicación y comportamiento de propietario.  
**CTA:** Aprende más / Solicita inspección GRATIS  
**Tono:** Educativo, empático, sin presión de venta

### 📱 Guión 9:16 — Vertical (Reels / TikTok)

> *Specs: 1080×1920 px · 60fps · Subtítulos centrados al 80% inferior · Logo top-center*

```
¿Sabías que la mayoría de los daños en el hogar pasan desapercibidos?

El problema crece silenciosamente hasta que es demasiado tarde.

Nuestros expertos lo detectan antes de que se convierta en una crisis.

Solicita tu evaluación GRATUITA hoy. Sin costo, sin compromiso.
```

### 🖥 Guión 16:9 — Horizontal (Facebook / YouTube)

> *Specs: 1920×1080 px · 30fps · Subtítulos en banda inferior · Logo top-left*

```
Muchos propietarios descubren los daños cuando ya es tarde.

Detectar el problema a tiempo puede ahorrarte miles de dólares.

[Tu Empresa] ofrece evaluaciones GRATUITAS para proteger tu hogar.

Sin costo. Sin compromiso. ¡Contáctanos hoy!
```

## MOFU — Middle of Funnel — Consideración

**Audiencia:** Prospectos tibios: interactuaron con TOFU, visitaron el perfil o buscaron el servicio.  
**CTA:** Habla con un experto / Solicita tu evaluación  
**Tono:** Informativo, profesional, generador de confianza

## 🔊 Integración ElevenLabs (Audio Automático)

El módulo detecta automáticamente si ElevenLabs está configurado:

```bash
# .env.example — si esta línea existe, se activa el modo audio
ELEVENLABS_API_KEY=
ELEVENLABS_VOICE_ID=Rachel   # opcional, default: Rachel
```

| Condición | Comportamiento |
|-----------|----------------|
| `ELEVENLABS_API_KEY` presente en `.env.example` | Genera guión + convierte a MP3 + sube a R2 + devuelve `audioUrl` |
| No presente | Solo devuelve guión como texto (`narration`), sin audio |

```
// Response con ElevenLabs activo
scripts.vertical_916.audioUrl = "https://r2.domain.com/campaigns/{id}/audio/916.mp3"
```

DETECCIÓN DE ELEVENLABS (CRÍTICO)
- Al inicializar el módulo, leer el archivo .env.example del proyecto
- Si existe la línea ELEVENLABS_API_KEY= (con o sin valor), activar el modo audio
- El use-case debe inyectar ElevenLabsPort de forma OPCIONAL usando @Optional()
- Si ElevenLabsPort está disponible:
    · Generar el guion como texto (campo: narration)
    · Convertir el narration a audio MP3 via ElevenLabsPort
    · Subir el MP3 a R2 con key: campaigns/{campaignId}/audio/{format}.mp3

  format: '9:16' | '16:9' | 'both',

  language?: string              // Default: 'es'

Response: CampaignResult {

  funnelStage: FunnelStage,

  productionNotes: ProductionNotes

  overlayTexts: string[],        // Textos para overlay en pantalla

}

Scene {
  timecode: string,
  title: string,
  visualDescription: string,
  stockSuggestion: string,
  durationSeconds: number
}

ProductionNotes {
  specs916?: string,
  specs169?: string,
  musicTone: string,
  colorPalette: string[],
  transitionStyle: string
}

    BOFU: urgente, directo, orientado a acción inmediata
    LOYALTY: cálido, agradecido, comunitario

- Guion adaptado dinámicamente a: businessName, niche, location, phone
